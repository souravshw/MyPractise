Understanding Climbing Stairs in Java

Learn how to count distinct ways to climb stairs using dynamic programming.

Real-World Analogy: The Step Combinations

To reach step N, you can either take 1 step from step N-1, or 2 steps from step N-2. Thus, the total ways to reach step N is the sum of ways to reach step N-1 and N-2. This is exactly the Fibonacci sequence, which we can solve in O(n) time using O(1) space.

Step-by-Step Execution Walkthrough

For N=5: Ways(1)=1, Ways(2)=2. For step 3: 1+2=3. For step 4: 2+3=5. For step 5: 3+5=8.

Java Implementation Code

Below is the complete, self-contained Java source code that solves this problem. It also includes a main method that traces the execution with console outputs.

package io.practise.dsa;

public class ClimbingStairs {

    // DP (Fibonacci style) - O(n), O(1) space
    public int climbStairs(int n) {
        if (n <= 2) return n;
        int first = 1, second = 2;
        for (int i = 3; i <= n; i++) {
            int third = first + second;
            first = second;
            second = third;
        }
        return second;
    }

    public static void main(String[] args) {
        ClimbingStairs solver = new ClimbingStairs();
        int n = 5;

        System.out.println("--- Climbing Stairs Demonstration ---");
        System.out.printf("Number of steps: %d\n", n);
        System.out.println("Distinct ways: " + solver.climbStairs(n));
    }
}

Conclusion

Solving the Climbing Stairs problem highlights how we can leverage key data structures and algorithmic paradigms (like two pointers, hash maps, or dynamic programming) to significantly optimize runtime and simplify code complexity.
