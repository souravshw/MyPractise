Understanding Non-overlapping Intervals in Java

Learn how to find the minimum intervals to remove to make the rest non-overlapping by sorting by end times.

Real-World Analogy: Maximizing Movie Screenings

To fit the maximum number of movies in a theater, you should always choose the movie that finishes earliest. This leaves the maximum room for subsequent movies. Any movie that overlaps with the chosen movie must be removed.

Step-by-Step Execution Walkthrough

Sort intervals by end times. Keep track of the end time of the last scheduled interval. For each next interval, if its start time is less than the current end time, increment the count of removals. Otherwise, update end time.

Java Implementation Code

Below is the complete, self-contained Java source code that solves this problem. It also includes a main method that traces the execution with console outputs.

package io.practise.dsa;

import java.util.Arrays;

public class NonOverlappingIntervals {

    // Sort by end time - O(n log n)
    public int eraseOverlapIntervals(int[][] intervals) {
        if (intervals.length == 0) return 0;
        Arrays.sort(intervals, (a, b) -> a[1] - b[1]);
        int end = intervals[0][1], count = 0;
        for (int i = 1; i < intervals.length; i++) {
            if (intervals[i][0] < end) {
                count++;
            } else {
                end = intervals[i][1];
            }
        }
        return count;
    }

    public static void main(String[] args) {
        NonOverlappingIntervals solver = new NonOverlappingIntervals();
        int[][] intervals = {{1, 2}, {2, 3}, {3, 4}, {1, 3}};

        System.out.println("--- Non-overlapping Intervals Demonstration ---");
        System.out.println("Input: " + Arrays.deepToString(intervals));
        System.out.println("Min intervals to remove: " + solver.eraseOverlapIntervals(intervals));
    }
}
